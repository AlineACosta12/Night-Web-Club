# Night Web Club

## Name
**Night Web Club** — a horror movie fan club website.

## Description
Night Web Club is a responsive, client-side website designed for horror movie enthusiasts. It acts as an online “club” where users can explore horror-related content, read news about upcoming releases, view curated recommendations, play a horror quiz, browse a gallery, and join discussions in a forum-style environment. :contentReference[oaicite:0]{index=0}

The site was designed with a horror theme in mind, using a dark visual identity and clear navigation so users can quickly access each section of the club. :contentReference[oaicite:1]{index=1}

### Features
- **Multi-page navigation**: Home, About the Club, News, Quiz/Gallery, Top 10, Events, Forum, About Us, Contact Us :contentReference[oaicite:2]{index=2}  
- **Horror-themed design**: black/red/white/grey colour palette and horror-style typography :contentReference[oaicite:3]{index=3}  
- **Responsive layout** using modern CSS layout approaches (grid/flexbox principles) :contentReference[oaicite:4]{index=4}  
- **Quiz** to make the experience more interactive :contentReference[oaicite:5]{index=5}  
- **Gallery** showing club/member cosplay images :contentReference[oaicite:6]{index=6}  
- **Forum/discussion space** for chatting about events and horror topics :contentReference[oaicite:7]{index=7}  
- **Contact page** with a contact form and email contact options :contentReference[oaicite:8]{index=8}  


## Installation

### Requirements
- A modern web browser (Chrome/Edge/Firefox)
- (Recommended) VS Code + Live Server extension for smooth navigation between pages

### Steps
1. Clone/download the repository.
2. Open the project folder in VS Code.
3. Run the site:
   - **Option A (simple):** open `index.html` in your browser.
   - **Option B (recommended):** right-click `index.html` → **Open with Live Server**.

## Support
If something doesn’t display correctly:
- Ensure you’re opening `index.html` from the project root.
- Use Live Server to avoid broken relative links between pages.
- Check the browser console for missing file paths (images/scripts/styles).
